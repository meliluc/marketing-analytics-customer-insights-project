\# Marketing Analytics \& Customer Insight Dashboard



Proyecto de Marketing Analytics que integra Data Analysis + Data Science + BI (Colab + Power BI).



\## Estructura

data/

raw/ # datos crudos (NO versionar)

interim/ # transformaciones intermedias
